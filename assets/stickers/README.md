# ملاحظة: ملفات الملصقات

هذا المجلد يحتوي على صور الملصقات التي يجمعها الأطفال.

## الملفات المطلوبة:

يجب إنشاء 20-30 صورة ملصقات بأسماء:
- `sticker_1.png` إلى `sticker_30.png`

## المواصفات:
- **Format**: PNG مع شفافية (Alpha channel)
- **Size**: 200x200 pixels
- **Style**: رسومات كرتونية ملونة وجذابة للأطفال
- **Themes**: 
  - حيوانات لطيفة
  - نجوم وكواكب
  - فواكه وخضروات
  - أشكال هندسية ملونة
  - شخصيات كرتونية

## كيفية الحصول على الملصقات:

### الخيار 1: توليد بالذكاء الاصطناعي
استخدم أدوات مثل:
- DALL-E
- Midjourney
- Stable Diffusion

**Prompt مقترح:**
```
cute cartoon sticker, colorful, simple design, 
child-friendly, transparent background, 
kawaii style, vector art
```

### الخيار 2: مواقع مجانية
- [Flaticon.com](https://www.flaticon.com/) - أيقونات مجانية
- [Freepik.com](https://www.freepik.com/) - رسومات مجانية
- [Icons8.com](https://icons8.com/) - أيقونات وملصقات

### الخيار 3: رسم يدوي
استخدم برامج مثل:
- Adobe Illustrator
- Inkscape (مجاني)
- Canva

## مثال على التسمية:
```
sticker_1.png  - قطة لطيفة
sticker_2.png  - نجمة ذهبية
sticker_3.png  - تفاحة حمراء
sticker_4.png  - صاروخ فضائي
...
sticker_30.png - قوس قزح
```

## بعد إضافة الملصقات:
قم بتشغيل:
```bash
flutter pub get
flutter run
```

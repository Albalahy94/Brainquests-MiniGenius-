# ملاحظة: ملفات الأصوات

هذا المجلد يحتوي على المؤثرات الصوتية للألعاب.

## الملفات المطلوبة:

1. **click.mp3** - صوت النقر على الأزرار
2. **success.mp3** - صوت الإجابة الصحيحة
3. **fail.mp3** - صوت الإجابة الخاطئة
4. **win.mp3** - صوت الفوز بالمستوى
5. **star.mp3** - صوت جمع النجمة

## كيفية الحصول على الأصوات:

### الخيار 1: مواقع مجانية
- [Freesound.org](https://freesound.org/) - أصوات مجانية بترخيص Creative Commons
- [Zapsplat.com](https://www.zapsplat.com/) - مؤثرات صوتية مجانية
- [Mixkit.co](https://mixkit.co/free-sound-effects/) - أصوات مجانية للاستخدام التجاري

### الخيار 2: توليد بالذكاء الاصطناعي
يمكنك استخدام أدوات AI لتوليد الأصوات

### الخيار 3: تسجيل يدوي
استخدم أي برنامج تسجيل صوت

## المواصفات:
- **Format**: MP3
- **Sample Rate**: 44100 Hz
- **Bit Rate**: 128 kbps
- **Duration**: 0.5-2 ثانية لكل صوت
- **Size**: أقل من 50 KB لكل ملف

## بعد إضافة الأصوات:
قم بتشغيل:
```bash
flutter pub get
flutter run
```
